Here are some of my projects

  • Omni Rental

    Html | CSS

    OmniRental is an eco-friendly car rental website offering a dynamic, user-friendly interface with responsive design for desktop, tablet, and mobile devices. Built with HTML and CSS, it features sections like Features, Pricing, and Testimonials, emphasizing secure services and customizable options.   • User Posts

    Html | CSS | JavaScript

    This webpage dynamically fetches and displays user information and their posts using the JSONPlaceholder API. Built with HTML, CSS, and JavaScript, it retrieves user details from one API and displays them in interactive user cards. Clicking on a user redirects to a separate page where their posts are dynamically fetched and displayed based on their ID. The implementation ensures seamless interaction and real-time data updates for an engaging user experience.

  • Online Library

    Html | CSS | JavaScript

    This online library features 5 sections for browsing books, built with HTML, CSS, and JavaScript. The books page dynamically displays books and allows users to sort by price (low/high) or rating. JavaScript fetches and renders book details (title, price, rating) from a simulated API. The content updates seamlessly as users interact with the sorting options, offering an engaging and responsive browsing experience.
